XMLWorker Specific CSS

  • page-break-before and page-break-after are supported but only with value always. When using page-break-before and page-break-after inside tags that are one element in PDF (like lists or tables) the outcome of adding a new page is unpredictable.
  • Parsed HTML tables can be spread over several pages. To implement repeating headers and/or footers, set
    • repeat-header:yes
    • repeat-footer:yes
    on the tag table as css style.
